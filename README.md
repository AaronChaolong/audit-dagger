# 开发背景说明  
本人在天健会计师事务所工作多年，主要从事财务审计工作。针对工作实务中的一些需求，尝试开发自动化工具来提高效率。最初，我使用的是vba工具，但vba代码是跟着工作簿的，且现有的xlsx格式文档不允许添加vba代码，原来的xls格式文档可以。因为vba代码工具使用不方便，后面通过Visual Studio开发office加载项。加载项安装后在excel或word界面下显示为菜单栏图标，使用很方便。  
本着开源的精神，现在我尝试通过GitHub网站分发并更新维护这一工具。我将该工具命名为“审计之刃”，我会在GitHub上上传项目源码，有需要的小伙伴可以自行研究，也可以向我反馈工具使用中发现的问题或提出优化建议。另外，考虑到大部分小伙伴没兴趣研究源码，只希望直接使用，我会将项目代码生成安装包，上传到GitHub项目库中，有需要的小伙伴直接下载exe或msi格式的安装包到电脑桌面，双击安装后即可。安装后，首次打开excel或wps，会提示导入加载项，确认接受。用户端使用界面如下图所示，会在菜单栏新增“审计之刃”，点击后可展开功能区。功能区分为通用功能区和项目功能区，通用功能区开发一些各项目或日常工作中通用的功能，项目功能区按照审计客户划分，一个集团客户做一个项目功能区，开发涉及该集团客户项目中的特性化的工具。

## 1. 通用功能区-月计划高亮名字  
我所在的天健业务部门，工作计划安排是按照月度发布的，发布后会经常修改更新。如下面截图所示，左侧是天数，右侧列示项目名称和项目组成员，比如员工“雍正”在16行、19行、21行都有项目安排，其他人也涉及多项目安排。截图只截取了本月安排的一部分，下面还有很多，有90多行。查看自己及查看他人的月度项目安排情况，很不方便。  
![月计划高亮功能说明用截图](./images/月计划高亮功能说明用截图.png)
问题解决思路：如果用鼠标点击某一人名，如左键单击“雍正”，计划表中所有的“雍正”都以某颜色高亮显示，这样查看计划安排就很方便了。  
### 使用说明
- 拿到发布的月度计划表（xls格式的工作簿）后，打开（用excel或wps都可以）计划表。如工作簿中存在多个sheet表，应切换至月计划这张sheet表。
- 点击功能区图标，弹出对话框提示“VBA代码已成功写入工作表！”，选择“确定”或直接关闭对话框。
- 此时，左键单击任一人名，计划表中该人名均以黄色高亮显示。已执行该功能的工作簿，下次打开时不用再重新执行，高亮功能依然会存在。
